Principais formatações

Negrito / Destaque

Nesta frase, tenho termo em destaque usando a tag strong (semântica)

Italíco / Ênfase

Nesta frase, temos um termo ítalico usando a tag i (não semântica)

Nesta frase, temos um termo em ênfase usando a tag em (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag mark

Texto grande e pequeno

Usavamos a tag big para Texto grande que agora foi movida para o CSS que agora usa-se a propriedade font-size

Para texto pequeno usamos a tag small

Texto deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido, mas não considerado, para isso usamos a tag del

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicinado depois, usamos a tag ins

Existe também o sublinhado com a tag u (não semântica)

Texto sobrescrito

Para inserir algo como x 20+3 usamos tag sup

Texto subscrito

Para inserir algo como H 2O usamos a tag sub

Outras formatações

Código-fonte

Para deixar a frase ou escrita em fonte de código como por exemplo window.document.getElementById('exemplo') usa-se a tag code

Pré-formatação

Usamos a tag pré para fazer as quebras de linha de um código seguindo o exemplo abaixo

Sem tag pre

function negrito() { let texto = document.getElementById("texto"); texto.style.fontWeight = (texto.style.fontWeight === "bold") ? "normal" : "bold"; } function italico() { let texto = document.getElementById("texto"); texto.style.fontStyle = (texto.style.fontStyle === "italic") ? "normal" : "italic"; } function sublinhado() { let texto = document.getElementById("texto"); texto.style.textDecoration = (texto.style.textDecoration === "underline") ? "none" : "underline"; }

Com a tag pre

function negrito() {
      let texto = document.getElementById("texto");
      texto.style.fontWeight = (texto.style.fontWeight === "bold") ? "normal" : "bold";
    }

    function italico() {
      let texto = document.getElementById("texto");
      texto.style.fontStyle = (texto.style.fontStyle === "italic") ? "normal" : "italic";
    }

    function sublinhado() {
      let texto = document.getElementById("texto");
      texto.style.textDecoration = (texto.style.textDecoration === "underline") ? "none" : "underline";}
